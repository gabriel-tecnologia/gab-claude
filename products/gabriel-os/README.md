# Gabriel OS

A única aba necessária para a investigação de um crime.

## Visão

Ser a única aba no navegador que um operador e um investigador da polícia civil
precisam para operar câmeras e investigar crimes, tornando-se a melhor ferramenta
de análise criminal do Brasil.

## Valor para o Cliente

- Acesso imediato a 15.000+ câmeras Gabriel (SP, RJ, Niterói, BH)
- Investigação autônoma sem depender de intermediários
- Redução do tempo de análise de dias para segundos
- Ferramenta gratuita para polícia, sem necessidade de licitação
