# Catálogo de Produtos

Produtos são a visão de negócio - o que entregamos de valor para clientes e stakeholders.

## Relacionamento com Tribes

- **1 Tribe → N Products**: Cada tribe é responsável por um ou mais produtos
- **Squads operam assets**: Squads dentro da tribe operam os repositórios/serviços que compõem o produto

## Como criar um novo produto

1. Crie uma pasta em `products/` com nome lowercase-hifenizado
2. Use o template em [templates/product.md](../templates/product.md)
3. Preencha visão, métricas e stakeholders
4. Atualize esta tabela
