# App Gabriel

Aplicativo mobile para cidadãos.

## Visão

Fazer com que seja óbvio a importância da Gabriel para um morador, de forma individual. Ser um ponto de consulta de informação granular sobre segurança pública em tempo real.

## Valor para o Cliente

- Informação sobre a seu bairro
- Visualização em tempo real
- Notificações de alertas
- Controle de dispositivos
- Histórico de eventos
