# Integrações

Conectores e APIs para integração com sistemas externos.

## Visão

Permitir que o ecossistema Gabriel se conecte de forma confiável com sistemas de autoridades primeiramente. Poderão ser usados com clientes enterprise também.

## Valor para o Cliente

- Disponibilização de inteligência da Gabriel de forma segura
- Conexão com autoridades policiais
