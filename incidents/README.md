## Quick Reference

### Severidades

| Severidade | Impacto | Tempo de Resposta |
|------------|---------|-------------------|
| **P1** | Sistema fora do ar | 15 min |
| **P2** | Funcionalidade crítica degradada | 1h |
| **P3** | Bug impactando usuários | 4h |
| **P4** | Bug menor | Próximo sprint |
