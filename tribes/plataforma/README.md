# Tribe Plataforma

Infraestrutura, dados, financeiro, contratos e operações de campo. Backoffice da empresa.

## Missão

Fornecer a infraestrutura, dados e sistemas internos que suportam todas as operações da Gabriel.

## Squads

| Squad                                                 | Foco                            |
| ----------------------------------------------------- | ------------------------------- |
| [Infraestrutura](./squads/infraestrutura/)            | Cloud, DevOps, SRE              |
| [Finanças e Contratos](./squads/financa-e-contratos/) | Billing, contratos, cobrança    |
| [Dados](./squads/dados/)                              | Data engineering, analytics     |
| [Operações de Campo](./squads/operacoes-de-campo/)    | Sistemas para técnicos em campo |

## Responsabilidades

- Infraestrutura cloud (AWS/GCP)
- Pipelines de dados
- Sistemas financeiros e de billing
- Ferramentas para operações de campo
- Observabilidade (DataDog)
