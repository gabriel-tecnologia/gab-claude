## Como usar

1. Copie o template em `templates/tribe-decision.md`
2. Crie um novo arquivo nesta pasta: `YYYY-MM-DD-titulo-da-decisao.md`
3. Preencha o template
4. Atualize o índice acima
5. Abra PR para revisão

## Escopo

Decisões de tribe afetam **múltiplos squads** e requerem aprovação do Engineering Manager + PM.

Para decisões de squad (escopo local), use o Linear ou README do squad.
