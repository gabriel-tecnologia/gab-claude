# Tribes

Tribes são a estrutura organizacional - como os times estão organizados.

## Estrutura

| Tribe                                     | Foco                                                    |
| ----------------------------------------- | ------------------------------------------------------- |
| [Borda](./borda/)                         | Hardware 2ª e 3ª geração, serviços de vídeo e placas    |
| [Plataforma](./plataforma/)               | Infra, dados, financeiro, contratos, operações de campo |
| [Produtos Digitais](./produtos-digitais/) | App mobile, Gabriel OS, Integrações                     |
